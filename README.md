Violence Reporting System
📌 Overview
The Violence Reporting System is a secure, web-based platform designed to help survivors and witnesses of violence report incidents confidentially. It provides:
- A safe reporting form for submitting detailed incident information.
- A tracking system to check the status of submitted reports.
- An admin dashboard for authorized personnel (PICs) to manage, assign, and resolve cases.
The system emphasizes privacy, safety, and accountability.

🚀 Features
- Submit a Report: Survivors or witnesses can securely submit incident details, attach evidence, and provide contact information.
- Check Status: Users can track the progress of their case using a unique tracking code.
- Admin / PIC Dashboard: Authorized personnel can log in, review reports, update statuses, and manage case assignments.
- Secure Authentication: Admin login with CSRF protection and hashed passwords.
- Evidence Uploads: Support for image, PDF, and document uploads.
- Responsive UI: Clean, modern design with a white + purple theme.

🛠️ Tech Stack
- Backend: PHP 8.x
- Database: MySQL (configured via XAMPP, port 3307 recommended)
- Frontend: HTML5, CSS3, Bootstrap 5, Font Awesome
- Security: CSRF tokens, password hashing (password_hash / password_verify
